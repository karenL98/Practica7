Frontend

Frontend es la parte de un programa o dispositivo a la que un usuario puede acceder directamente. Son todas las tecnologías de diseño y desarrollo web que corren en el navegador y que se encargan de la interactividad con los usuarios. Para cualquiera que no trabaje directamente con el desarrollo web, conceptos como desarrollo front-end o back-end pueden ser difíciles de entender. Apile todos los marcos, lenguajes de programación, API y bibliotecas que los desarrolladores de aplicaciones para el usuario utilizan para crear aplicaciones, y puede resultar muy confuso.

Para ayudarlo a convertirse en un profesional en todo lo relacionado con el desarrollo de front-end y back-end, hemos escrito una serie de artículos que cubren lo que necesita saber sobre los fundamentos de esta área del desarrollo de front-end y back-end, de alguna manera los no desarrolladores pueden entender.

El desarrollo de front-end y back-end Dos palabras clave para comprender cómo funcionan los sitios web que quizás haya escuchado antes son el frontend y el backend. A menudo se habla de ellos (y se enseñan) por separado y se ven como dos cosas muy diferentes, pero son esenciales y están estrechamente conectados en la mayoría de los proyectos de desarrollo web. La interfaz, también llamada programación " del lado del cliente ", es lo que sucede en el navegador. Es todo lo que el usuario ve e interactúa. El back-end, también llamado programación " del lado del servidor ", ocurre en el servidor y la base de datos. Es la maquinaria que trabaja detrás de escena para potenciar esas características sofisticadas con las que los usuarios interactúan en el lado del cliente.
